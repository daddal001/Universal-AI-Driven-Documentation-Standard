# Example Documents

> Real, filled-in examples showing "what good looks like" when following these documentation standards.

## Purpose

Templates show you the structure. **Examples show you the quality.**

Use these examples to:

- See what a complete, filled-in document looks like
- Understand the level of detail expected
- Copy and adapt for your own project
