# Security Policy

## Scope

This repository contains **documentation standards and templates** — no executable code, no runtime services, no secrets. Security concerns here relate to:

- Incorrect security guidance in standards (e.g., bad advice in `24-SECURITY_COMPLIANCE.md`)
- Accidentally committed credentials in examples or templates
- Supply chain concerns with CI workflow templates

## Supported Versions

| Version | Supported |
|---------|-----------|
| 1.x.x  | Yes       |
| < 1.0   | No       |
